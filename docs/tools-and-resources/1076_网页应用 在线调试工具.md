# 网页应用 在线调试工具

在线调试工具 可以帮助开发者**实时调试**运行中的网页应用，更方便地定位问题，提高解决问题的效率。
调试能力支持的范围：
- 操作系统：目前支持Android 以及鸿蒙系统
- 飞书版本号：Android需要v5.1及以上版本，鸿蒙需要v7.54及以上版本
- 网络环境：需要支持外网访问（访问chrome服务的必须要求）
