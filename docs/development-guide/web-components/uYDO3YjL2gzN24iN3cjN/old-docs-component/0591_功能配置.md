# 功能配置

你可以通过组件功能配置接口来操作组件的表现。

## 使用方式
```js
// 通过render方法渲染组件会返回组件实例
myComponent = window.webComponent.render('DocsComponent', config, node)

// 通过setFeatureConfig方法修改组件的配置属性
myComponent.config.setFeatureConfig({
  // 详细参数请参考 FEATURE & Options
  HEADER: {
    enable: false, // 隐藏头部
  },
});
```

## FEATURE & Options

maxWidth | number | 缩放图片时最大宽度 | &nbsp;

shareLink | boolean | 是否显示分享链接 | &nbsp;
shareMethod | boolean | 是否显示分享方式 | &nbsp;
MODAL | object | 模态窗 | `支持`
outerMask | boolean | object | 有模态窗时， 组件外是否有遮罩 | &nbsp;
enable | boolean | 是否显示 | &nbsp;
zIndex | number | 组件外遮罩层级 | &nbsp;
HEADER | object | 头部 | `部分支持`
enable | boolean | 是否显示头部 | `支持`

enable | boolean | 是否显示 | `支持`
open | boolean | 是否展开局部评论侧栏 只在初始化设置有效 | &nbsp;
global | object | 全文评论 | &nbsp;
enable | boolean | 是否显示 | `支持`
SIDEBAR | object | 右侧边栏(doc1.0生效) | `不支持`

DIRECTORY | object | 目录 | `不支持`

makeCopyEnable | boolean | 是否显示创建副本 | &nbsp;
