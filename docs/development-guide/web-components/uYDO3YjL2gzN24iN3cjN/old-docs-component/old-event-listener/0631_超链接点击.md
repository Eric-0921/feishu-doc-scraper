# 超链接点击
监听用户点击正文中的超链接，并且拦截超链接的跳转打开。
## 示例
```js
myComponent.event.onHyperlinkClick(function(url) {
  // ...
});
```

## 返回
|属性|	类型|	说明|
| ---|----- | ------- | 
|url|String |超链接地址
